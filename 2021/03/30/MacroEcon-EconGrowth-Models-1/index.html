经济增长理论研究的主要问题：

  • 存在性问题：即一个国家的经济是否存在着一种长期增长的可能性？
  • 稳定性问题：即一个国家的经济是否存在着一种稳定增长的可能性？
  • 恢复性问题：即一个国家的经济偏离正常的增长轨道，怎样才能恢复？

哈罗德—多马模型

哈罗德-多马模型提出于 20 世纪 40 年代，它将凯恩斯的分析加以长期化和动态化。

哈罗德-多马模型的假设条件

  1. 全社会只生产一种产品，这种产品既可以作为消费品，又可以作为资本品；
  2. 储蓄 S 是产出 Y 的函数，即 S = sY ，这里的 s 代表社会的储蓄比例，即储蓄在产出中所占有的份额；
  3. 生产过程中只使用两种生产要素，即劳动与资本。这两种生产要素为固定技术系数，即它们在生产中的比率固定，不能相互替代
  4. 劳动力按照一个固定不变的比率增长；
  5. 不存在技术进步，即生产技术水平是既定的，也不存在资本折旧问题（哈罗德中性假设）；
  6. 生产规模报酬不变，即生产任何一单位产品所需的资本和劳动的数量固定不变。

基本增长方程式

  • $G_A$ ：一个国家的经济增长率，$G_A = \Delta Y / Y$ ;
  • $s$：一个国家的储蓄率，$s = S/Y$；
  • $v$：一个国家的资本-产量比，$v = K/Y$ 。

推导：

1.在哈罗德中性假设（边际资本-产量比等于原有资本-产量比）下：$\Delta K / \Delta Y = K/Y = v$；若折旧等于零，即 $I = \Delta K$。则有：$I/\Delta Y = v => I = v \cdot\Delta Y$ 。

2.假设储蓄 S 是产出 Y 的函数，则：$ S = s Y$。

3.宏观经济均衡条件：$I = S$。

经济含义：要实现均衡的经济增长，一个国家的经济增长率必须等于储蓄率与资本-产量比之比。

有保证的增长方程

  • $G_I$ ：有保证的经济增长率 ;
  • $s$：合意的储蓄率，即一个国家的居民希望保持的储蓄水平；
  • $v$：合意的资本-产量比，即一个国家的厂商希望保持的资本-产量比。

经济含义：要保证均衡的经济增长，一个国家的储蓄率与资本-产量比必须符合居民与厂商的意愿。

自然增长方程式

  • $G_N, s_n, v_n$ ：充分就业时的经济增长率、储蓄率和资本-产量比 ;
  • $n$：人口增长率。

经济含义：要实现充分就业，一个国家的经济增长率必须等于该国人口增长率。

哈罗德—多马模型对基本问题的回答

1.存在性问题：$G_A = G_I = G_N = n$

2.不稳定性问题

  • 若 $G_A > G_I =G_N$ ，则 $s >s_i$ 或 $v<v_i$，结果 $s_i$ 进一步收缩或 $v_i$ 进一步扩大，从而使 $G_A$ 更加偏离 $G_I$ 和 $G_N$ 。

  • 若 $G_A < G_I =G_N$ ，则 $s < s_i$ 或 $v>v_i$ ，结果 $s_i$ 进一步扩大或 $v_i$ 进一步收缩，从而使 $G_A$ 更加偏离 $G_I$ 和 $G_N$ 。

3.可调控原理

  • 从短期看，政府应运用财政政策和货币政策使 $G_A$ 与 $G_I$ 保持一致，避免经济增长的波动；

  • 从长期看，政府应运用产业政策、分配政策和人口政策使 $G_A, G_I$ 与 $G_N$ 保持一致，以实现国民经济持续、稳定的增长。

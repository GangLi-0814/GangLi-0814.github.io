宏观经济常用数据库介绍。

宏观经济变量（4+2） $\rightarrow$ 宏观经济变量的数据？

  • 经济增长：GDP
  • 充分就业：失业率
  • 物价稳定：CPI，GDP deflator， PPP
  • 国际收支平衡（资产价格的稳定）：顺差、逆差及外汇储备来衡量
  • 资产价格稳定：房地产？
  • 收入分配：基尼系数？
  • 人口
  • 财政

层面：世界、全国、区域

频率：月度、季度和年度

数据来源：

国际：

U.S. macroeconomic data - American Economic Association

  • World Bank Open Data: https://data.worldbank.org/
    • ssc install wbopendata
    • https://github.com/jpazvd/wbopendata
    • World Development Indicators database
  • IMF：https://www.imf.org/en/Data
    • “世界经济展望”(World Economic Outlook)
  • FRED（联邦储备经济数据库）
    • Stata: ssc install freduse
  • UN Commodity Trade Statistics Database；UN Comtrade联合国商品贸易统计数据库
  • WTO
  • WHO
  • UNdata
  • Penn World Tables

国内：

  • CCER数据库
  • CEIC
  • RESSET宏观经济数据库
  • CSMAR数据库
  • 中华人民共和国国家统计局
  • 中国人民银行统计数据
  • 东方财富网经济数据中心
  • 和讯网宏观数据
  • 新浪财经-中国宏观经济数据
  • 中经网数据库
  • 中宏统计数据库
  • 塔塔统计数据库
    • 《塔塔统计数据库》由塔塔资讯提供。其数据主要来源：国家统计局、国家海关总署、工商总局、税务总局；中国工业和信息化部、发改委、文化部、教育部、商务部等部门；中国证监会、中国保监会、中国银监会及各个行业协会；塔塔资讯自有的信息采集系统、市场调查机构、研究机构等。
  • EPS数据库
  • 中国研究数据服务平台（Chinese Research Data Services，简称CNRDS）
  • Choice金融终端

Python包：

  • Tushare:https://waditu.com/document/2?doc_id=25
    • 适用对象：量化投资分析师（Quant）、对金融市场进行大数据分析的企业和个人、开发以证券为基础的金融类产品和解决方案的公司、正在学习利用Python进行数据分析的人
  • AKShare:https://www.akshare.xyz/introduction.html

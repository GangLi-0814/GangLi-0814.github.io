Markdown 简介

何为 Markdown ?

简介

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

优势
  • 轻量易上手，不像 LaTeX 需要搭建环境；
  • 颜值高，使用 Typora 编辑器更换主题；
  • 排脱格式困扰，沉浸式写作。

可以用来做什么？

  • 记笔记、做攻略
  • 公众号创作、Readme
  • 沉浸式写作，需要文字的地方都能用到…
